AboutÂ¶

PyVista is a helper module for the Visualization Toolkit (VTK) that takes a different approach on interfacing with VTK through NumPy and direct array access. This package provides a Pythonic, well-documented interface exposing VTKâ€™s powerful visualization backend to facilitate rapid prototyping, analysis, and visual integration of spatially referenced datasets.

This module can be used for scientific plotting for presentations and research papers as well as a supporting module for other mesh dependent Python modules.

  • discretize: Discretization tools for finite volume and inverse problems. discretize provides toVTK methods that return PyVista versions of their data types for creating compelling visualizations.

  • pymeshfix: Python/Cython wrapper of Marco Atteneâ€™s wonderful, award-winning MeshFix software.

  • tetgen: Python Interface to Hang Siâ€™s C++ TetGen Library

Getting StartedÂ¶

If you have a working copy of VTK, installation is simply:

$ pip install pyvista

You can also visit PyPi or GitHub to download the source.
